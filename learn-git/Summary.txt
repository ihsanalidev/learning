- note : remote repository = github | local repository = laptop
- If you already have a project locally but not yet on a remote, here are example steps :
git remote add origin https://github.com/ihsanalidev/learn-git.git
git branch -M main
git push -u origin main
- If you already have a project on a remote but not yet on your local machine, here are the steps to get started:
open repo in github
click "<>code"
copy link ssh
in terminal run this code : git clone copy-paste-link-ssh
(this will copy and connect remote to local machine)
